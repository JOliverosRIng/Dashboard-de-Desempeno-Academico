# Dashboard de Desempeño Académico

Dashboard interactivo desarrollado en **Power BI** para analizar el rendimiento académico de estudiantes por asignatura, permitiendo visualizar indicadores clave, detectar estudiantes en riesgo y comparar el desempeño entre materias.

---

## Características

- 📈 KPIs académicos:
  - Promedio del curso
  - % de aprobación
  - % de reprobación
  - Nota máxima y mínima
  - Estudiantes en riesgo

-  Visualizaciones interactivas:
  - Histograma de distribución de notas
  - Gráfico de niveles de desempeño
  - Mapa de calor por estudiante y asignatura
  - Diagramas de caja y bigotes

-  Segmentación dinámica por materia.

## Visualizaciones incluidas

### 1. Distribución de notas
Permite analizar cómo se distribuyen las calificaciones de los estudiantes en diferentes rangos.

### 2. Nivel de desempeño
Clasificación del rendimiento académico en:
- Alto
- Medio
- Bajo
- Crítico

### 3. Mapa de calor
Visualización de notas por estudiante y asignatura mediante colores para identificar patrones de rendimiento.

### 4. Caja y bigotes
Análisis estadístico de:
- Mediana
- Dispersión
- Valores atípicos
- Variabilidad entre materias

---

## Objetivo del proyecto

Apoyar el análisis académico y la toma de decisiones educativas mediante visualizaciones intuitivas y métricas clave que permitan:

- Detectar estudiantes en riesgo.
- Comparar desempeño entre materias.
- Identificar tendencias académicas.
- Facilitar el seguimiento del curso.

## Posibles mejoras futuras

- Análisis predictivo con Machine Learning.
